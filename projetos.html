Os projetos que participei até o momento se consistem em trabalhos/projetos dentro da própria faculdade propostos em várias matérias diferentes. Os mais recentes em que participei são: Um gerenciador de vacinas online com ideia principal de funcionar para as pessoas verificarem as vacinas que já tomaram e agendar vacinas em postos de saúdes mais próximos delas. Com uma aplicação web usando framework Laravel e junto disso integrado a um app mobile tamnbém, que no caso a parte mobile seria exclusivamente para o "paciente" verificar e agendar suas vacinas enquanto a web iria gerenciar também a parte administrativa do projeto(no caso seria implantada nos postos de saúde parceiros).

Um app mobile de gerenciamento de comanda para restaurantes que a finalidade seria substituir o cardápio dando mais praticidade ao cliente.

Um mini crm feito em laravel com finalidade de aprender a trabalhar com pelo menos o básico do framework.

Um site bem simples utilizando java web com o framework bootstrap também com a finalidade de aprender a trabalhar com o framework e também a linguagem java web.
